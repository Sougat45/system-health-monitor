## 🔍 About

**System Health Monitor** is a powerful yet lightweight Bash script that provides real-time monitoring of your Linux system's vital statistics. It displays CPU usage, memory consumption, disk space, running processes, and network information—all in a clean, auto-refreshing terminal interface.

Perfect for:
- 🎓 Students learning Linux
- 💼 System Administrators
- 🖥️ Server Monitoring
- 📊 Quick System Diagnostics

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔄 **Real-time Monitoring** | Auto-refreshes every 5 seconds |
| 💻 **CPU Usage** | Shows current CPU utilization percentage |
| 🧠 **RAM/Memory** | Displays Total, Used, and Free memory |
| 💾 **Disk Space** | Shows disk usage of root partition |
| ⚡ **Top Processes** | Lists top 5 CPU-consuming processes |
| 🌐 **Network Info** | Displays system IP address |
| 📅 **System Info** | Shows date, time, hostname, and username |
| 🪶 **Lightweight** | No external dependencies required |
| 🐧 **Cross-Distro** | Works on Ubuntu, CentOS, Fedora, Kali, etc. |
